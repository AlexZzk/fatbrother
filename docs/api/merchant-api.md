# 商户模块 接口文档

> 云函数名：`merchant`
> 负责 Sprint：S2
> 状态：待开发

## 接口列表

| Action | 说明 | 鉴权 |
|--------|------|------|
| verifyInviteCode | 验证邀请码 | 是 |
| apply | 提交入驻申请 | 是 |
| getApplyStatus | 查询申请状态 | 是 |
| getMerchantInfo | 获取商户信息 | 部分 |
| updateSettings | 更新店铺设置 | 是(商户) |
| toggleStatus | 切换营业状态 | 是(商户) |
| getNearbyList | 获取附近商家 | 否 |

> 详细接口文档将在 Sprint 2 开发时补充。
