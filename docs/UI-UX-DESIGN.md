## 一、设计规范（Design System）

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 简洁高效 | 摆摊场景下商户操作需快速，C端点餐路径需最短 |
| 信息层级清晰 | 价格、状态等关键信息一眼可见 |
| 操作反馈即时 | 每个操作都有明确的视觉/触觉反馈 |
| 双端风格统一 | C端和B端使用同一套设计语言，仅色彩做区分 |

### 1.2 色彩体系

#### 主色

| 色值 | 用途 | 说明 |
|------|------|------|
| `#FF6B35` | C端主色 | 暖橙色，食欲感，用于按钮、高亮、Tab选中 |
| `#1677FF` | B端主色 | 蓝色，专业感，用于商户后台按钮、状态标识 |

#### 辅助色

| 色值 | 用途 |
|------|------|
| `#00B578` | 成功/营业中/已完成 |
| `#FF3B30` | 错误/关店/取消/删除/价格强调 |
| `#FF9500` | 警告/待处理/倒计时 |
| `#8C8C8C` | 禁用/置灰 |

#### 中性色

| 色值 | 用途 |
|------|------|
| `#222222` | 主标题文字 |
| `#333333` | 正文文字 |
| `#666666` | 次要文字（描述、备注） |
| `#999999` | 辅助文字（提示、占位符） |
| `#CCCCCC` | 分割线、边框 |
| `#F5F5F5` | 页面背景色 |
| `#FFFFFF` | 卡片/容器背景 |

#### 渐变色

| 用途 | 渐变值 |
|------|--------|
| C端主按钮 | `linear-gradient(135deg, #FF6B35, #FF8F5E)` |
| B端主按钮 | `linear-gradient(135deg, #1677FF, #4A9AFF)` |

### 1.3 字体规范

使用微信小程序系统默认字体栈，无需引入自定义字体。

| 层级 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|---------|
| H1 | 36rpx | Bold (700) | 1.4 | 页面大标题（极少使用） |
| H2 | 32rpx | Bold (700) | 1.4 | 区域标题、卡片标题 |
| H3 | 28rpx | Medium (500) | 1.4 | 小标题、列表项标题 |
| Body | 28rpx | Regular (400) | 1.5 | 正文内容 |
| Caption | 24rpx | Regular (400) | 1.4 | 辅助说明文字 |
| Small | 22rpx | Regular (400) | 1.3 | 标签、角标 |
| Price-L | 36rpx | Bold (700) | 1.0 | 大号价格展示（结算页） |
| Price-M | 30rpx | Bold (700) | 1.0 | 中号价格（商品卡片） |
| Price-S | 24rpx | Bold (700) | 1.0 | 小号价格（规格加价） |

### 1.4 间距系统（基于 8rpx 网格）

| 变量 | 值 | 使用场景 |
|------|------|---------|
| spacing-xs | 8rpx | 紧凑间距（标签内间距） |
| spacing-sm | 16rpx | 小间距（文字间距、图标与文字） |
| spacing-md | 24rpx | 中间距（卡片内边距、列表项间距） |
| spacing-lg | 32rpx | 大间距（区域间距、页面内边距） |
| spacing-xl | 48rpx | 超大间距（区域分隔） |

### 1.5 圆角规范

| 层级 | 值 | 使用场景 |
|------|------|---------|
| radius-sm | 8rpx | 小元素（标签、小按钮） |
| radius-md | 16rpx | 卡片、输入框 |
| radius-lg | 24rpx | 弹窗、底部面板 |
| radius-full | 999rpx | 圆形（头像、胶囊按钮） |

### 1.6 阴影规范

| 层级 | 值 | 使用场景 |
|------|------|---------|
| shadow-sm | `0 2rpx 8rpx rgba(0,0,0,0.06)` | 卡片默认阴影 |
| shadow-md | `0 4rpx 16rpx rgba(0,0,0,0.1)` | 悬浮元素（购物车栏） |
| shadow-lg | `0 8rpx 32rpx rgba(0,0,0,0.15)` | 弹窗、模态框 |

### 1.7 基础组件规范

#### 按钮

| 类型 | 高度 | 圆角 | 说明 |
|------|------|------|------|
| 主按钮（大） | 88rpx | radius-full | 全宽，用于页面底部提交操作 |
| 主按钮（中） | 72rpx | radius-full | 常规操作按钮 |
| 主按钮（小） | 56rpx | radius-full | 内联操作（如"接单"） |
| 次要按钮 | 同上 | 同上 | 白底 + 主色边框 + 主色文字 |
| 文字按钮 | 自适应 | 无 | 纯文字，主色 |
| 危险按钮 | 同上 | 同上 | 红色系，用于拒单、取消 |
| 禁用按钮 | 同上 | 同上 | 灰色背景 `#CCCCCC`，不可点击 |

#### 卡片

```
┌─ Card ──────────────────────┐
│  padding: 24rpx              │
│  background: #FFFFFF         │
│  border-radius: 16rpx        │
│  box-shadow: shadow-sm       │
│  margin-bottom: 24rpx        │
└──────────────────────────────┘
```

#### 输入框

| 属性 | 值 |
|------|------|
| 高度 | 80rpx |
| 内边距 | 0 24rpx |
| 边框 | 1rpx solid #CCCCCC |
| 圆角 | radius-md |
| 聚焦边框 | 主色 |
| 占位符颜色 | #999999 |

#### 标签（Tag）

| 类型 | 背景 | 文字 | 圆角 |
|------|------|------|------|
| 默认 | #F5F5F5 | #666666 | radius-sm |
| 选中 | 主色10%透明度 | 主色 | radius-sm |
| 状态-成功 | #E8F8EE | #00B578 | radius-sm |
| 状态-警告 | #FFF3E0 | #FF9500 | radius-sm |
| 状态-错误 | #FFEBEE | #FF3B30 | radius-sm |

| 价格实时计算 | 每次选项变化时重新计算：基础价 + SUM(各选项加价) |
| 点击"加入购物车" | 加入成功，弹窗关闭，底部购物车数量+1，抛物线动画 |
| 数量调整 | [-]到1时继续减不生效（最小1），通过购物车弹窗可删除 |

## 四、全局交互与导航流程

### 4.1 C端 TabBar 配置

| Tab | 图标 | 文字 | 页面路径 |
|-----|------|------|---------|
| 1 | 首页图标 | 首页 | /pages/index/index |
| 2 | 订单图标 | 订单 | /pages/order/list |
| 3 | 我的图标 | 我的 | /pages/mine/index |

- 选中态：图标和文字变为主色 `#FF6B35`
- 未选中：图标和文字为 `#999999`

### 4.2 登录拦截策略

| 场景 | 是否需要登录 |
|------|-------------|
| 浏览首页、商家列表 | 否 |
| 进入商家详情、浏览菜单 | 否 |
| 加入购物车 | 否 |
| 提交订单/支付 | 是（弹出授权登录） |
| 查看订单列表 | 是 |
| 进入"我的"页面 | 否（展示未登录态） |
| 进入商户管理 | 是 |

### 4.3 全局 Loading 与错误处理

**Loading：**
- 页面级加载：页面中心展示 loading 动画 + "加载中..."
- 操作级加载：按钮变为 loading 状态（转圈 + 文字变灰），防重复点击
- 列表加载：底部展示 loading 指示器

**错误处理：**
- 网络错误：Toast 提示"网络异常，请稍后重试"
- 业务错误：Toast 展示具体错误信息
- 空数据：展示对应空状态插画 + 引导文案

**Toast 规范：**
- 位置：屏幕居中偏上
- 时长：1.5秒自动消失
- 类型：成功（绿色对勾）、失败（红色叉号）、提示（蓝色感叹号）

### 4.4 核心用户流程

#### 流程1：用户点餐下单

```
首页
 → 点击商家卡片
 → 商家详情页（浏览菜单）
 → 点击[+]选规格
 → 规格弹窗选择 → 加入购物车
 → 重复选购...
 → 点击购物车"去结算"
 → 订单确认页（填备注）
 → 点击"提交订单"
 → [未登录则弹出授权登录]
 → [请求订阅消息授权]
 → 调起微信支付
 → 支付成功 → 结果页
 → 查看订单详情（等待商家接单）
```

#### 流程2：商户处理订单

```
进入商户后台
 → 看到"待处理订单"提醒 / 收到新订单弹窗
 → 进入订单管理
 → 查看订单详情（商品+备注）
 → 点击"接单" → 开始制作
 → 制作完成 → 点击"出餐完成"
 → 用户取餐 → 点击"确认完成"
 → 订单结束，触发分账
```

#### 流程3：商户入驻

```
"我的"页面 → 点击"商户管理"
 → 入驻引导页
 → 点击"我要入驻开店"
 → 填写邀请码 + 店铺信息
 → 提交申请
 → 等待审核
 → 审核通过 → 进入商户后台
 → 设置菜单 → 开始营业
```

### 4.5 页面转场动画

| 转场类型 | 动画 |
|---------|------|
| 普通页面跳转 | 从右向左滑入（系统默认） |
| 返回 | 从左向右滑出（系统默认） |
| 弹窗弹出 | 从底部向上滑入 + 遮罩淡入 |
| 弹窗关闭 | 向下滑出 + 遮罩淡出 |
| TabBar切换 | 无动画（即时切换） |

## 五、适配与边界情况

### 5.1 安全区域适配

- 底部固定栏（购物车栏、提交栏）需适配 iPhone 底部安全区域
- 使用 `padding-bottom: env(safe-area-inset-bottom)` 或微信 `safeArea` API

### 5.2 长文本处理

| 场景 | 处理方式 |
|------|---------|
| 商家名称 | 最多显示1行，超出省略号 |
| 商品名称 | 最多显示1行，超出省略号 |
| 商品描述 | 最多显示2行，超出省略号 |
| 店铺公告 | 首页卡片1行截断，详情页全部展示 |
| 规格摘要 | 购物车中1行截断，如"大杯/中辣/珍珠/椰..." |
| 订单备注 | B端完整展示，C端列表1行截断 |

### 5.3 图片规范

| 场景 | 建议尺寸 | 压缩 |
|------|---------|------|
| 商家头图 | 750x400px | 质量80%，限制200KB |
| 商家头像 | 200x200px | 质量80%，限制50KB |
| 商品图片 | 400x400px | 质量80%，限制100KB |
| Banner | 750x340px | 质量80%，限制200KB |

### 5.4 骨架屏

以下页面在数据加载中时展示骨架屏：
- 首页商家列表
- 商家详情页菜单区域
- 订单列表

骨架屏使用浅灰色块 `#EEEEEE` + 微闪光动画模拟内容占位。
